１．高速化に配慮したサーバー機能

  • 高速化：ストレージはソリッドストレージ（SSD）
  • 高速化：「WEXALR Page Speed Technology」
  • 高速化：HTTP/2対応
  • 高速化：超高速アプリケーション実行環境(LSAPI)

高速化面では「SSD」「LSAPI」 「コンテンツキャッシュ」「HTTP2」が導入されていて他の高速化対応されているレンタルサーバーとあまり変らない内容になっています。

さらにAI技術である「WEXAL」が導入されたことにより大幅なスピードアップを実現。サーバー処理速度が業界最高峰となっています（2020年12月自社調べ）。

２．安定性をもたらすサーバー体制

  • Webサーバーは同時アクセスの処理に優れたNginx（エンジンエックス）
  • RAID10での管理体制

Webサーバーソフトは「Xginx（エンジンエックス）」となっていて、同時の大量アクセス処理に強くなっています。

管理体制はRAID10となっていて何か問題が発生したときでも復旧に優れた体制となっています。

３．データ容量は400GBと大容量

データ容量は400GBと充分な容量が用意されています。お名前サーバーには1つのプランしかないため、容量は400GBで固定されます。

４．稼働率はSLA99.99％で安定

稼働率はSLA保証制度として99.99％を数値として設定されています。 これは、稼働率が規定数値（99.99％）を下回った月は一定金額の返金をしてくれる制度になります。

また、実際の稼働率としても99.99％を実現させています（2021年度6月調べ）、かなり安定して管理されているレンタルサーバーだと言えるでしょう。

７．テスト環境の構築ができる

ワードプレスの制作にあたり、テスト環境を作って管理することが出来ます。 ワードプレスのバージョンアップ時やプラグインを導入する際にテスト環境で確認すれば、エラーが出てもすぐに戻せるため、エラーを回避することが出来ます。

共用サーバーにおいてこのテスト環境のサービスを実施しているのは「CPI」「さくらインターネット」の2社だけになります。
