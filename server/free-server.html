【2023年】ワードプレスも作れる！無料レンタルサーバーおすすめ３選

更新：2023年7月13日

無料で借りられるレンタルサーバーとして人気のサービスを探してみました。

このページではエックスサーバーフリー、スターサーバーフリー、XREAFREEの3つの無料サーバーについて詳しく解説していきます。

まず、無料サーバーの特徴を説明すると機能に制限があること。 例えばディスク容量が数GBまで落とされている点が挙げれます。

いくつかの制限がありながらも通常プランと変わらずワードプレスも作れますので、 練習用に借りるプランとしては充分に機能するでしょう。

それぞれの無料サーバーでは幾つかのプランが提供されていますが、今回はWordpressが作れる無料プランに限定して 比較・解説していきたいと思います。

ちなみに、これから紹介する完全無料のレンタルサーバーよりも費用はかかりますが、 独自ドメインを作って最安でワードプレスを作りたいのであればスタードメインがおすすめです。

スタードメインならドメイン取得するだけで無料サーバーがついてきます。実質ドメイン代だけでワードプレスが作れます。

コストは年間で1,000円ほどになります。広告表示はなし、容量は20GB、オールSSDの高速環境です。

→年額50円(税込)からの高機能ドメイン取得サービス、スタードメイン

その１．エックスフリー

「エックスフリー」は無料レンタルサーバーではありますが、3種類のプランを選択することが出来ます。

  1. HTMLサーバープラン
    広告なし。独自ドメイン10個分まで静的HTMLサイトが作れます。
  2. PHP・MySQLプラン
    スマホ表示時に広告あり。データベース5個分までWordpress作成可能
  3. Wordpressプラン
    スマホ表示時に広告あり。Wordpress簡単インストールあり。よりWordpressに最適化されています。

HTMLプランだけ広告表示が出ない仕様になっています。ただし多くの方が選ぶであろうWordpressプランについては スマホ画面のみ広告が表示されるようになります。

その２．スターサーバーフリー

スターサーバーフリーもエックスフリーと同様に3つのプランが用意されています。

  1. フリープラン
    広告なし。独自ドメイン1つ利用可能。静的HTMLサイトが作れます。
  2. フリー容量増加プラン
    スマホ表示時に広告あり。独自ドメイン1つ利用可能。容量4GBまでOK
  3. Mysqlプラン
    スマホ表示時に広告あり。MYSQL1つ利用可能。Wordpressが作成できる

「エックスフリー」と比べると、容量が多いプランがそろっています。 ただしWordpressを作る場合は「PHP Mysqlプラン」で提供されているデータベースが1個なので、ワードプレスの作成は1つのみとなります。

広告表示に関しては、エックスフリー同様に「フリー増量プラン」「フリーMysqlプラン」において スマホで見たときにだけ広告が表示されます。HTMLサイト専用となるフリープランにおいては広告表示はありません。

ウィークポイント

今回、3つの無料サーバーを紹介していますが、どのプランでもワードプレスを作ることが出来るため 練習をかねて作ってみたい人には打ってつけのサービスだと言えるでしょう。

しかし、いくつかの制限もありますので、どんなデメリットがあるのかについて合わせて紹介してみたいと思います。

データ容量が1GBや2GBに制限されている

例えばエックスフリーであれば容量が1GBまでになります。制限はされていますが、1GBあれば数十記事は作ることが出来ます。

その後に続けて記事を作りたい場合は、有料プランへの乗換えを検討することになります。 将来的な有料プランへの移行も考えている方は、エックスサーバーやスターサーバーなど乗換先の有料サーバーのサービスについても調べておいた方が良いでしょう。

メールアドレスが作れない（XREA Freeは除く）

エックスフリーとスターサーバーフリーではメールアカウントの作成は出来ません。

XREA Freeではメールアドレスも作成できます。

スマホの画面には広告が表示される

基本的にどの無料サーバーでも、スマホの画面にはページ内に広告が入ります。 Xfreeの場合は、次のようなバナー広告が画面に表示されることになります（PCの場合）。

広告が入らない無料プラン（HTMLプラン）もありますが、その場合データベースが0個のためワードプレスは作れません。

おすすめプラン

ワードプレスを作るなら「エックスフリー」のwordpressプラン

ワードプレスを作成したいと考えている場合、「エックスフリー」のWordpressプランがおすすめになります。 無料でありながら、データベースが5個迄使えるためワードプレスも5個作成することができます。

エックスサーバーのサービスも兼ねているため、管理画面やかんたんインストール機能もエックスサーバーと同様に出来るようになっています。

メールアドレスを作成したいなら「XREA Free」プラン

メールアドレスも作りたいという方におすすめなのが、メールアドレスも作れてしまうXREA Free。 メールアカウントを作成できる無料レンタルサーバーはXREA FREEだけなので、メールを考えているならこのプランにしましょう。

メールアドレスを作る場合、独自ドメインは持っていたいところなので、 ドメインをムームードメイン辺りで作っておくといいかもしれません。
